
# ForgeBot Ultimate — All‑in‑One Discord Server Bot (Slash Commands)

**Includes:** verification, tickets + transcripts, moderation suite, utilities, welcome/DM, logging, anti‑raid/anti‑spam, status rotator, reaction‑role buttons, simple giveaways.

## Quick Start
1. Put files on GitHub or unzip locally.
2. Create `.env` (copy from `.env.example`) and fill IDs and your token.
3. Install: `pip install -r requirements.txt`
4. Run: `python bot.py`

## Railway
- Start command: `python bot.py`
- Add environment variables from `.env.example`
